# Sadaqa Support Bot

Telegram bot for transparent sadaqa (charity) collections via Telegram Stars.

## Purpose

This project is created to:
- Collect sadaqa transparently
- Show active campaigns
- Visualize progress
- Keep donors anonymous
- Archive reports with proof

## Structure

- bot.py – main bot logic
- db.py – database structure
- ui.py – user interface and buttons
- requirements.txt – dependencies
- .env.example – environment variables example

## Status

Initial structure created. Deployment via Render planned.
